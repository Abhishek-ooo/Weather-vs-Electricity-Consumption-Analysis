# Weather vs Electricity Consumption Analysis

## Overview

This project analyzes the relationship between weather conditions and electricity consumption using Python, Pandas, NumPy, SQL, and MySQL.

## Technologies

- Python
- Pandas
- NumPy
- MySQL
- SQL
- VS Code

## Features

- Automatic dataset generation
- Relational database design
- SQL analysis
- Automated CSV import
- Weather-based electricity usage analysis

## Database Tables

- Buildings
- Weather
- Electricity Usage
- Sensors
- Electricity Tariff
- Maintenance Logs
